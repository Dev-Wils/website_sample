Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous . In a process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1 – h6 headings includes heading (level 1–6)
p paragraph include a paragraph of text
table table include a table
tr table include a row of data
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header header block element
div division divide block elements
ol ordered list order lists
ul unordered list unorder lists
li list list item
blockquote block quotation show formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophsitication</em> technical sophsitication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://www.learnenough.com">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/IMZAFuq" alt="Michael Hartl">
span span inline element <span>Call me Ishmael</span> Call me Ishmael

Document Tags

Tag Name Purpose
html Hyper Text Markup Language tells the browser this is an html document
head head contains the setting of the html document
body body contains the content of the html docuements
title title title of the html document
meta metadata contains infromation about the html document
